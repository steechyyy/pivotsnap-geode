# Pivot Snap!!

simple editor qol mod that enhance the quality of lives by allowing ppl to snap the pivot point to the sides and edges when warping objects

this is my first ever mod so i hope u enjoyed, dont forget to like, subscribe and hit that bell

# How to use

### **before use, check the snap mode in the mod's settings! <cr>IF YOU'RE ON MOBILE, THE "keybind" OPTION ISN'T SUPPORTED, so please keep it "button"**</c>

**The default keybind to snap is X.** You're able to change this in the mod settings. When using the button, **the button is found on the right side of the screen.**
